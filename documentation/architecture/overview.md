# BellyBox Architecture

## Overview
BellyBox is a hyper-local food subscription platform.

## Core Components
1. **Next.js Frontend**: App Directory, Server Components.
2. **Supabase Backend**: Auth, Database, Storage, Edge Functions.
3. **Razorpay**: Payment Gateway.

## Status
System is currently in **Phase 0** (Complete) and moving to **Phase 1** (Subscription V2).
